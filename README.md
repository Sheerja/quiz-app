# Quiz App

A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript.

## Features

* Multiple Choice Questions
* Score Tracking
* Correct and Wrong Answer Highlighting
* Progress Indicator
* Responsive Design
* Interactive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.

## Project Preview

Test your knowledge by answering multiple-choice questions and track your score in real time.
